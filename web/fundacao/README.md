# Fundação do Web

Este diretório reúne as decisões que precisam ser fechadas `antes do build principal`.

## Objetivo

Evitar começar a implementação com ambiguidades em:

- stack
- integrações
- branding
- direção visual
- composição das telas
- sinais de mercado que o produto precisa respeitar
